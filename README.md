# 🎄 LifeOS 2.0 - 个人生活操作系统

这是一个基于 Python Streamlit 开发的个人数字化生活管理系统。集成了状态追踪、任务管理、财务记录以及 AI 智能周报功能。

## ✨ 核心功能

* **🎅 氛围主题**：内置圣诞/生日专属 UI，带有动态下雪效果和倒计时。
* **📊 状态调控 (HUD)**：
    * 五维状态雷达图（情绪、认知、觉察、动机、人际）。
    * 实时显示“财富余额”与“专注时长”。
* **⚔️ 作战中心 (IPO)**：
    * 基于 IPO (Input/Process/Output) 模型的专注计时器。
    * 支持 6 大领域能力值积累（核心能力、创新实践等）。
* **🤖 AI 驾驶舱**：
    * 集成 DeepSeek AI，支持智能对话。
    * **自动周报**：一键读取本周数据，生成温馨幽默的总结周报。
* **🎁 积分商店**：通过专注赚取时间积分，兑换“快乐水”、“游戏时光”等奖励。

## 🛠️ 技术栈

* **前端/框架**: Streamlit
* **数据分析**: Pandas, Numpy, Matplotlib
* **数据库**: SQLite3 (本地轻量级存储)
* **AI 支持**: DeepSeek API

## 🚀 如何运行

1.  **安装依赖**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **启动应用**:
    ```bash
    streamlit run app.py
    ```

## ⚠️ 注意事项

* 初次运行时会自动生成 `life_os.db` 数据库文件。
* 使用 AI 功能需要在界面侧边栏输入 DeepSeek API Key。
* 建议上传代码时在 `.gitignore` 中忽略 `.db` 文件以保护隐私。
