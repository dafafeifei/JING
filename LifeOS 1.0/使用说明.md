项目名称：LifeOS 个人成长操作系统 (V11.0)
开发者：李景杰
开发环境：Python 3.14 + Streamlit

【如何运行】
由于本项目采用了最新的 Python 3.14 开发，建议使用源码方式运行以获得最佳兼容性：

1. 双击运行 [一键安装依赖.bat]
   - 系统将自动配置 Streamlit、Pandas、Matplotlib 等数据科学运行库。

2. 双击运行 [启动LifeOS.bat]
   - 系统将以“桌面应用模式”启动，自动隐藏浏览器地址栏，提供沉浸式体验。

【核心特性】
- 五维状态雷达 (基于 Matplotlib)
- 沉浸式专注计时器 (Session State 管理)
- AI 战略幕僚 (集成 DeepSeek API)
- 游戏化经济系统 (SQLite 数据持久化)
